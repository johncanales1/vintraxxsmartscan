Vintraxx API

OBD-II Vehicle Diagnostic Report System

Authentication

/api/v1/auth

POST /check-email Check if email exists
POST /send-otp Send OTP to email
POST /verify-otp Verify OTP code
POST /register Register new user (supports dealer flag)
POST /login Login existing user (supports dealer flag)

Scan Management

/api/v1/scan

POST /submit Submit OBD-II scan data Requires Auth
GET /report/:scanId Get report (poll for completion) Requires Auth
GET /history Get user's scan history Requires Auth

Dealer Portal

/api/v1/dealer

POST /login Dealer portal login
GET /profile Get dealer profile info Requires Auth
PUT /profile Update dealer profile (price per labor hour, logo) Requires Auth
GET /reports Get dealer's scan reports Requires Auth

Vehicle Appraisal

/api/v1/appraisal

POST /value Get vehicle valuation/appraisal

System

Health Check

GET /api/v1/health Health check endpoint
